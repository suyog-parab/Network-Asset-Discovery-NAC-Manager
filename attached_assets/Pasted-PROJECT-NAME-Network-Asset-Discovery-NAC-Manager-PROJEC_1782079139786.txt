PROJECT NAME

Network Asset Discovery & NAC Manager

⸻

PROJECT GOAL

Create a centralized network asset discovery and NAC management platform capable of:

1. Discovering network devices automatically.
2. Building a complete asset inventory.
3. Mapping devices to users.
4. Approving devices.
5. Syncing approved devices to FreeRADIUS.
6. Assigning VLANs dynamically through FreeRADIUS.
7. Moving unknown devices into Quarantine VLAN.
8. Managing Cisco Catalyst 2960X environments.
9. Supporting future multi-site deployments.
